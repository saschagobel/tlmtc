# Security Policy

Please do not report security vulnerabilities through public GitHub issues or
pull requests.

To report a vulnerability, use GitHub's private vulnerability reporting:

1. Open the repository's **Security and quality** tab.
2. Click **Report a vulnerability**.
3. Include enough detail to reproduce and assess the issue.

Useful reports include:

- the affected `tlmtc` version
- your Python version and operating system
- a minimal reproducible example
- the expected and actual behavior
- the potential security impact
